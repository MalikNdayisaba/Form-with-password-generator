User Information Form with Password Generator

This project is a web-based user information form that collects personal details and generates a secure password for account setup. The form includes client-side validation to ensure that users fill in all required fields correctly.

Features
Responsive HTML form for user data collection
JavaScript form validation:
Checks for empty first name, last name, and email
Validates proper email format
Password generator with options:
Lowercase letters
Uppercase letters
Numbers
Symbols
Styled using Tailwind CSS for a clean, modern look
Alerts the user on successful submission or missing/invalid fields
Technologies Used
HTML5
CSS (Tailwind)
JavaScript (ES6)
